# 🌐 Modern Bootstrap Website Project

## 📌 Project Overview

This project is a responsive multi-page website built using Bootstrap 5.  
The objective was to explore Bootstrap components, remix UI patterns creatively, and design clean, modern web pages.

The website includes:

- Home Page
- About Page
- Contact Page
- Fully responsive layout
- Modern UI styling
- GitHub deployment

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (via CDN)
- Git & GitHub
- GitHub Pages (for deployment)

---

## 🚀 Step-by-Step Development Process

### 1️⃣ Research Phase
- Explored Bootstrap 5 documentation.
- Reviewed Bootstrap example templates.
- Studied layout structure and component usage.
- Identified reusable UI patterns (navbar, hero, cards, forms).

### 2️⃣ Project Setup
- Created folder structure.
- Added index.html, about.html, contact.html.
- Linked Bootstrap 5 via CDN.
- Created external style.css file.

### 3️⃣ Homepage Development
- Built responsive navbar.
- Designed hero section with gradient background.
- Added feature section using Bootstrap grid.
- Created service cards with hover effects.
- Designed footer section.

### 4️⃣ About Page
- Added page header banner.
- Used two-column layout for about content.
- Added service highlight section.
- Maintained consistent styling.

### 5️⃣ Contact Page
- Designed responsive contact form.
- Used Bootstrap form classes.
- Added contact information section.
- Ensured layout works on mobile devices.

### 6️⃣ Styling Improvements
- Added hover animations to cards.
- Improved spacing using utility classes.
- Added rounded corners and subtle shadows.
- Maintained consistent color theme.

### 7️⃣ Testing & Responsiveness
- Tested on desktop, tablet, and mobile views.
- Verified navbar collapse functionality.
- Ensured no horizontal scrolling issues.

### 8️⃣ Deployment
- Initialized Git repository.
- Pushed project to GitHub.
- Enabled GitHub Pages for live hosting.

## ⚠️ Challenges Faced

### 1. Navbar collapse not working
Problem: Bootstrap JS bundle missing.  
Solution: Added Bootstrap bundle CDN at bottom of page.

### 2. Card heights inconsistent
Problem: Uneven card sizes.  
Solution: Used `h-100` class to equalize height.

### 3. Spacing inconsistencies
Problem: Sections looked crowded.  
Solution: Used `py-5`, `mb-4`, and custom `.section-padding`.

### 4. Deployment confusion
Problem: GitHub Pages initially not loading.  
Solution: Set correct branch (main) under Pages settings.
