# ModernSite - Bootstrap 5 Web Builder Project

![ModernSite Banner](https://via.placeholder.com/1200x400)

## 🚀 Project Overview
**ModernSite** is a responsive, modern, and professional website built using **Bootstrap 5**, designed for web startups or digital agencies.  
The project demonstrates **component remixing**, clean UI design, and a polished, mobile-first layout.  

This project includes **three fully responsive pages**:  
- **Home**: Hero section, Features, Services, Testimonials, Call-to-Action  
- **About**: Company mission, team, services, and expertise  
- **Contact**: Contact form, contact information, and embedded map  

---

## 🛠 Technologies Used
- **HTML5** – Semantic and structured markup  
- **CSS3** – Custom styling and hover/animation effects  
- **Bootstrap 5** – Layout, components, utilities, responsive grid  
- **JavaScript** – Bootstrap JS for interactive components (carousel, navbar)  
- **Git & GitHub** – Version control and hosting  
- **GitHub Pages / Netlify** – Live deployment
